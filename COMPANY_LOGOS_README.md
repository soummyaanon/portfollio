# Company Logos

To add company logos to your experience section, place PNG/JPG images in the `public/company-logos/` directory with these names:

- `cardiovascular-institute-of-orlando.png`
- `cloudoplus.png`
- `chatsguru.png`

The avatars will automatically show the company initials if no logo is found.

## Example:
```
public/
  company-logos/
    cardiovascular-institute-of-orlando.png
    cloudoplus.png
    chatsguru.png
```
